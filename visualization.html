Variable Relationships¶

The variables used to select the girls that make up the STEM group are the explanatory variables in this project. They are:

  • grade in math
  • grade in science
  • whether or not the girl had trouble completing homework
  • decision-making tendency
  • fact-gathering tendency
  • pre-analytical tendency
  • systematic judging tendency
  • post-analytical tendency

Please refer to the codebook (http://hlhodges.github.io/stem_girls/research_question.html) for greater detail.

The following code bins the response variables into two categories: 1, representing high, and 0, representing low.

data2 = pandas.read_csv("addhealth_pds.csv", low_memory = False)
## convert response variables back to numeric ##
for x in all_variables:
    data2[x].convert_objects(convert_numeric = True)

## convert sex variable to numeric 
data2['BIO_SEX'] = data2['BIO_SEX'].convert_objects(convert_numeric = True)
## remove boys' data ##
data2.drop( data2[data2["BIO_SEX"] != 2].index, axis = 0, inplace = True )				
## remove irrelevant variables ##
data2.drop( [x for x in data2.columns if x not in all_variables], axis = 1, inplace = True )

## re-binning categorical response variables into two categories ##

## high self-esteem variables defined in terms of "agree" or "strongly agree" responses
def re_bin_val (val):
    if val["H1PF30"] < 3:
        return 1
    else:
        return 0

def re_bin_appr (val):
    if val["H1PF33"] < 3:
        return 1
    else:
        return 0

def re_bin_judg (val):
    if val["H1PF34"] < 3:
        return 1
    else:
        return 0    
    
data2["H1PF30"] = data2.apply( lambda val: re_bin_val (val), axis = 1)
data2["H1PF33"] = data2.apply( lambda val: re_bin_appr (val), axis = 1)
data2["H1PF34"] = data2.apply( lambda val: re_bin_judg (val), axis = 1)

## low difficulty response and high relationship interest defined in terms of 
## "agree"/"strongly agree" and "very much"/"quite a bit," respectively ##
def re_bin_diff (val):
    if val["H1PF15"] in [ 4, 5 ]:
        return 1
    else:
        return 0

def re_bin_rela (val):
    if val["H1ID6"] in [ 4, 5 ]:
        return 1
    else: 
        return 0

data2["H1PF15"] = data2.apply( lambda val: re_bin_diff (val), axis = 1)    
data2["H1ID6"] = data2.apply( lambda val: re_bin_rela (val), axis = 1)

## high romantic interest (in both males and females) defined in terms of "yes" responses
def re_bin_fem (val):
    if val["H1NR1"] == 1:
        return 1
    else:
        return 0
def re_bin_mal (val):
    if val["H1NR2"] == 1:
        return 1
    else:
        return 0
data2["H1NR1"] = data2.apply( lambda val: re_bin_fem (val), axis = 1)
data2["H1NR2"] = data2.apply( lambda val: re_bin_mal (val), axis = 1)

The following code recodes and re-bins the explanatory variables, removing unusable responses (skip, don't know, etc.). Refusals to respond are retained (coded equal to zero); the notable lack of refusals from girls in the STEM group suggests that exploring this response might provide insight.

Math grade, science grade, and difficulty with homework are combined in the variable Academic Performance and recoded such that higher numerical scores represent higher math and science grades and less difficulty with homework.

The other five explanatory variables are combined into the numerical variable Cognitive Style, which is coded such that higher scores represent stronger endorsement of a cognitive style that is assumed to be advantageous to STEM success.
